Return Value

object

A string representation of the property value specified by the column mapping name if the conversion is successful; otherwise, an empty string.

Remarks
 
The method checks if the value is a GridCell and if its column is a GridTextColumn. It then retrieves the property value from the data context using the column's mapping name. If the mapping name is null or empty, or if the property cannot be found, the method returns an empty string.
