RecordNavigationBar Class

Namespace: Qhta.SF.WPF.Tools
Assembly: Qhta.SF.WPF.Tools (in Qhta.SF.WPF.Tools.dll)
Navigation bar for navigating through records in a SfDataGrid. It provides commands to navigate to the first, previous, next, and last records. It also displays the current record number and the total number of records in the grid. It should be placed under the SfDataGrid.
