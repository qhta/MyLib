UndoMgr Class

Namespace: Qhta.UndoManager
Assembly: Qhta.UndoManager (in Qhta.UndoManager.dll)
Manager class for Undo/Redo functionality in application. Records actions and manages undo/redo stacks. Only registered actions can be undone or redone.
